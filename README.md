# Sport Hub
A sports fixtures and match details app built with React, TypeScript, Vite, and Tailwind CSS.

## Prerequisites

- Node.js 18+ (recommended: 20 or 22)
- npm (comes with Node.js)

## Install dependencies

From the project root:

```bash
npm install
```

## Run the project

Start the development server:

```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`) in your browser.

## Other scripts

 Command        | Description                          

 `npm run build`  Type-check and build for production 
 `npm run preview`  Serve the production build locally 
 `npm run lint`  Run ESLint                          

## Tech stack

- React 19 + React Router 7
- Vite 7
- TypeScript
- Tailwind CSS
